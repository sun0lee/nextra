### Nextra v4 업데이트 주요 사항

Nextra v4가 이제 출시되었으며, 여러 가지 중요한 개선 사항과 새로운 기능들이 포함되었습니다:

- 🚀 **App Router 지원**: Next.js의 App Router로 전면 전환, Pages Router는 더 이상 사용되지 않으며 최신 Metadata API를 지원합니다.
- ⚡ **빠른 페이지 로딩**: 최적화된 번들 크기와 성능 향상으로 웹사이트 응답 속도가 빨라졌습니다.
- 🔍 **새로운 검색 엔진**: Rust 기반의 Pagefind를 통해 더 정확하고 효율적인 검색 경험을 제공합니다.
- 🏎️ **Turbopack 지원**: 개발 경험을 개선하고, 커뮤니티의 오랜 요구사항을 해결합니다.
- 🌍 **RSC + i18n**: 국제화(i18n) 지원 강화, 다국어 웹사이트 개발이 더 편리해졌습니다.
- 🎨 **UI 스타일 최적화**: UI 디자인을 개선하여 시각적 일관성을 강화했습니다.
- 📱 **모바일 최적화**: 작은 화면 장치에 대한 디스플레이와 상호작용 경험을 최적화했습니다.
- 📚 **강력한 디렉토리 구조**: 새로운 "Page Map"이 md/mdx 및 app/ 디렉토리 내의 jsx/tsx 페이지를 수집할 수 있습니다.
- 🛠️ **원격 MDX 렌더링**: 임의의 콘텐츠 소스에서 MDX를 로드하고 올바르게 사이드바 네비게이션을 생성합니다.
- ✨ **강화된 TOC**: Markdown 제목을 정확하게 표시하며, JSX, 코드 블록, 수학 공식 등도 지원합니다.

자세한 정보는 [Nextra v4 공식 업그레이드 가이드](https://the-guild.dev/blog/nextra-4)를 참조하세요.

### Tailwind CSS v4 업데이트 주요 사항

Tailwind CSS v4가 공식 출시되었으며, 여러 중요한 개선 사항과 새로운 기능들이 포함되었습니다:

- 🚀 **새로운 고성능 엔진**: 빌드 속도가 최대 5배 향상되었고, 증분 빌드 속도가 100배 이상 향상되었습니다.
- 💎 **최신 웹 디자인을 위한 지원**: 최신 CSS 기능을 기반으로 구성, 예를 들어 cascade layers, `@property`를 사용하는 커스텀 속성 및 `color-mix()` 등이 포함됩니다.
- 🎯 **간소화된 설치 과정**: 의존성이 줄어들고, 설정이 거의 필요 없이 CSS 파일에 한 줄 코드만 추가하면 됩니다.
- 📦 **공식 Vite 플러그인**: 성능을 극대화하고 설정을 최소화합니다.
- 🔍 **자동 콘텐츠 감지**: 모든 템플릿 파일을 자동으로 감지하여 별도 설정 없이 사용됩니다.
- 📂 **내장된 CSS 파일 묶기 지원**: 추가 도구 없이 여러 CSS 파일을 묶을 수 있습니다.
- 🛠️ **CSS 우선 설정**: 개발자 경험을 개선하여, CSS 파일 내에서 직접 프레임워크를 커스터마이즈하고 확장할 수 있습니다.

자세한 내용은 [Tailwind CSS v4 업그레이드 가이드](https://tailwindcss.com/docs/upgrade-guide)와 [공식 블로그](https://tailwindcss.com/blog/tailwindcss-v4)를 참조하세요.
