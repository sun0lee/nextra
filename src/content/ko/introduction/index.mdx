Next.js + Nextra + TypeScript + Tailwind CSS로 구축된 무료, 오픈소스이면서 강력한 템플릿


# 설치 및 실행 

* 의존성 설치

```bash
pnpm install
```

* 로컬개발

```bash
pnpm run dev
```
