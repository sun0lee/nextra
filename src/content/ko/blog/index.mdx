 제도 비교 분석 Matrix

##  1.  할인율 산출 방식
  - • 기준 수익률 출처 (무위험 수익률 + 스프레드 구조)
  - • 유동성 프리미엄/일반 프리미엄 적용 여부
  - • 일관성 유지 방식 (locking 여부)
  - • 통화별 차별화 여부

##  2.  보험부채의 리스크 반영 방식
  - • 리스크 조정(Risk Adjustment) 포함 여부 및 방식
  - • 위험률 변동성 반영 방법 (VaR, CTE 등)
  - • 시나리오 수/수준 (1-in-200, 99.5% 등)

##  3.  계약서비스마진(CSM)/잔여마진 설정
  - • 최초 산출 방식
  - • 이자율 사용 방식 (초기 vs 현행 EIR)
  - • 손실계약 처리 방식
  - • 업데이트 주기 및 방법

##  4.  수익성 지표 및 PL 인식 구조
  - • 보험서비스 PL 구성
  - • 금융손익 구분 방식 (OCI 포함 여부)
  - • 시가평가 손익 인식 시점 (예실차는 줄이는게 좋을까 ??)
